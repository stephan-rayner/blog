# Agents

## Writing Guidelines

- Never use this character —
- All titles should be capitalized using title casing (see section on Title Case Description)
- Write in a style consistent with a Grafana Labs repository
- Write in a style consistent with this repository

### Title Case Description

Title case rules vary by style guide but generally require capitalizing the first and last words of a title, along with all major words (nouns, pronouns, verbs, adjectives, adverbs, and subordinating conjunctions). Minor words such as articles (a, an, the), coordinating conjunctions (and, but, or), and short prepositions are typically lowercase unless they appear at the beginning or end of the title.
